## Introduction
Ziggy is an open source platform that uses AI to help gather feedback by automatically conducting unstructured feedback interviews.

## Tech Stack
Here is what is used to build Ziggy.

- Next
- TypeScript
- Redis via Upstash
- Tailwind CSS
- Vercel
- Postgres via Supabase
- Prisma
- NextAuth
- And more...
